En este ejercicio debo crear un array bidimensional con varios artículos del contenido de un pedido guardada en un carrito de compra de un sitio web.

De cada artículo del pedido debe tener los siguientes campos:

    Descripción

    cantidad

    precio

    Disponibilidad

El objetivo del ejercicio es crear un Script en JavaScript que muestre el contenido del carrito en forma de tabla de solo los articulos 
que estan disponible con el importe total solo de los articulos que se encuentran disponibles.
